flujo de trabajo) está en el [README raíz](../README.md) y en

**https://TU-PROYECTO.vercel.app** ← reemplazar por la URL real del proyecto en Vercel.

## Scripts

```bash
npm ci                 # instalar dependencias
npm run dev            # desarrollo
npm run lint           # ESLint
npm run test           # tests unitarios (Vitest)
npm run coverage       # cobertura
npm run test:e2e       # tests E2E (Playwright)
npm run build          # build de producción
npm run preview        # servir el build (puerto 4173)
```
